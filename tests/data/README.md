
# Speech Audio Test Data

This directory contains test audio files for speech recognition. The files are used to verify the functionality of the speech recognition system.
The following lists file names and their source information:
- [`OSR_us_000_0034_8k.wav`](speech_audio/OSR_us_000_0034_8k.wav): Sample from the Open Speech Repository (OSR) dataset ([audio file `OSR_us_000_0034_8k.wav`](https://www.voiptroubleshooter.com/open_speech/american.html), [transcription `List 72`](https://www.cs.columbia.edu/%7Ehgs/audio/harvard.html)).
